#Simple AI Agent

**Description**
When a chat text is sent the AI agent uses its brain(LLM) which then uses the necessary tool according to the user input i.e fetch contact infomation from the contact database, send an eamil, schedule an event in the calendar. The agent then stores the information in simple memory.

**Nodes Used**
-Start Chat Trigger
-AI Agent: Brain- Deepseek chat, Memory- Simple memory, Tools- Contacts Database, Gmail, Google Calendar

**Use case**
Use when the events that are to happen are non-deterministic and an AI workflow is'nt suitable
