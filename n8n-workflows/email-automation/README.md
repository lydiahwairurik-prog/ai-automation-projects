#Email Automation Workflows
Workflows related to email routing, classification and processing

##Workflow: Customer Support Workflow
**Description**
This workflow receives an email, classifies the message to customer support where the message goes through an AI Agent where it drafts a response that is relevant using its knowledge base and a reply is sent or it is classified as other wher the workflow stops
**Nodes Used**
-Gmail Trigger 
-Text Classifier
-Open Router
-AI Agent
-Simple Vector Store
-Embeddings Cohere
-Gmail Reply a Message
**Use Case**
Customer support email classification and response automation
